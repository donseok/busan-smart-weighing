# 부산 스마트 계량 시스템 - 사용자 퀵 매뉴얼

**문서번호**: BSW-MAN-003 | **버전**: 1.2 | **최종 수정**: 2026-01-29

---

## 1. 로그인

| 구분 | 절차 |
|------|------|
| **웹 로그인** | 브라우저 접속 → 시스템 URL 입력 → ID/PW 입력 → **로그인** 클릭 |
| **📱 모바일 로그인** | 앱 실행 → ID/PW 또는 OTP 입력 → **로그인** 터치 |
| **비밀번호 분실** | 시스템 관리자에게 초기화 요청 |

> **잠금 정책**: 비밀번호 **5회 연속 실패** 시 계정 **30분 자동 잠금**

---

## 2. 운전자 퀵 가이드

### LPR 자동 계량 (정상 흐름)

| 단계 | 절차 |
|:----:|------|
| **1** | 계량대 진입 → 차량 자동 감지 |
| **2** | LPR 카메라 자동 촬영 → AI 차량번호 인식 |
| **3** | 배차 정보 자동 매칭 → 중량 자동 측정 |
| **4** | 계량 완료 → 전자 계량표 자동 생성 |
| **5** | ✅ **카카오톡/SMS**로 계량표 수신 확인 |

### 📱 모바일 OTP 계량 (LPR 실패 시)

| 단계 | 절차 |
|:----:|------|
| **1** | 전광판에 표시된 **OTP 번호 확인** |
| **2** | 모바일 앱 실행 → OTP 번호 입력 |
| **3** | 배차 목록에서 해당 건 선택 → 계량 진행 |
| **4** | ✅ 전자 계량표 확인 및 공유 |

### 📱 즐겨찾기 & 이력

| 기능 | 절차 |
|------|------|
| **즐겨찾기 등록** | 배차/업체 상세 화면 → ☆ 아이콘 탭 → ★로 변경되면 등록 완료 |
| **즐겨찾기 확인** | 홈 화면 → 즐겨찾기 영역에서 바로 접근 |
| **계량 이력 조회** | 홈 화면 → **이력 조회** → 기간별 과거 계량/배차 이력 확인 |

## 3. 담당자 퀵 가이드

| 업무 | 경로 및 절차 |
|------|-------------|
| **배차 등록** | 배차관리 → **신규등록** → 차량/품목/목적지 입력 → **저장** |
| **계량 조회** | 계량관리 → 날짜/차량번호 등 조건 검색 → 상세 보기 |
| **계량대 현황** | 계량대 현황 → 실시간 계량 진행 상태 모니터링 |
| **출문 승인** | 출문관리 → 대기 목록 확인 → **승인** 또는 **반려** 처리 |
| **통계 확인** | 통계 → 기간/조건 선택 → 조회 → **엑셀 다운로드** |
| **공지사항 관리** | 공지사항 → **신규등록** → 작성 → **저장** → **게시** / **고정(📌)** |

## 4. 공통 기능 퀵 가이드

| 기능 | 절차 |
|------|------|
| **알림 설정** | 마이페이지 → **알림 설정** → 유형별 수신 ON/OFF 전환 |
| **비밀번호 변경** | 마이페이지 → **비밀번호 변경** → 현재/새 비밀번호 입력 → **변경** |
| **테마 변경** | (웹) 우측 상단 사용자 메뉴 → 다크/라이트 모드 전환 |
| **문의하기** | 마이페이지 → **문의하기** → 물류관제실/자재창고 선택 → **전화 연결** |
| **공지 확인** | 공지사항 → 카테고리 필터/검색 → 상세 확인 (📌 고정 공지 상단 표시) |

## 6. 주요 주의사항

| 항목 | 내용 |
|------|------|
| OTP 유효시간 | **5분** 이내 입력 필수 (초과 시 재발급) |
| OTP 입력 제한 | 최대 **3회** 실패 시 재발급 필요 |
| 계량 중 차량 이동 | **절대 금지** (측정 오류 및 안전사고 위험) |
| 문제 발생 시 | 즉시 **물류관제실**로 연락 |

> **안전 수칙**: 계량대 위에서는 반드시 차량을 **완전 정지**하고, 전광판 안내에 따라 행동하십시오.
